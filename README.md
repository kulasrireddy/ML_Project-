# Daily Water Intake Prediction - Machine Learning Project

## Description
This project uses Machine Learning to analyze and predict daily water intake patterns based on a dataset. The model is trained using a Random Forest Classifier. The project includes data preprocessing, visualization, model training, prediction, and performance evaluation.

## Dataset
- File: Daily_Water_Intake.csv
- Contains information related to daily water consumption.

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Model
- Random Forest Classifier

## Evaluation Metrics
- Accuracy Score
- Classification Report (Precision, Recall, F1-score)

## How to Run
1. Install required libraries.
2. Open the notebook file.
3. Run all cells to train and test the model.
